A web app made using the Github API. It uses vanilla JS request method to get the JSON info from the API.

OpenWeatherMap API. JQuery getJSON method. Requires browser to have location enabled. If it's not, check the same app in the Angular section.

News Headers API. JQuery getJSON method and asynchronous loading for each news section.

Flickr Api. Using AJAX and each() so that the gallery images fade-in one by one.

NASA APIs, two in one. Using JQuery getJSON to display NASA's pic of the day and a list of objects that are "near" earth.

Wikipedia API. Using JQuery's Ajax method to fetch a list of your search results.

A simple quote generator API that also includes the possibility to Tweet whatever quote you get.

Using the Twitch API to showcase some channels in the featured list and the possibility to search for a specific channel.
